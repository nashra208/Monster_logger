Monsterlogger is a **Windows-based Python keylogger** built to understand how real-world attackers collect data.

This project was developed in a **controlled lab environment** to study OS-level events, user input handling, and data monitoring techniques commonly abused by malware.

## Functionalities
- **Captures keystrokes**  
- **Monitors clipboard data (text and images)**  
- **Extracts saved Wi-Fi SSIDs and passwords on Windows systems**
